## 第三部分：简要说明

### 改动目的

将主题切换按钮的功能从仅切换 CSS `dark` 类扩展为**完整联动**，点击切换按钮后以下所有视觉元素同步切换：

| 视觉元素 | 暗色模式 | 亮色模式 |
|----------|----------|----------|
| **主图背景** | `swiper_image_dark`（来自后端接口） | `swiper_image_light`（来自后端接口） |
| **星星颜色** | 白色 `#ffffff` | 黑色 `#070707` |
| **星星光晕** | 浅黄 `rgba(255,230,150,0.5)` | 浅蓝 `rgba(160,200,255,0.5)` |
| **网站 Logo** | `dark_logo` | `light_logo` |
| **导航文字** | 白色 `#fff` | 黑色 `#000` |
| **导航悬浮** | 灰色 `text-gray-500` | 灰色 `text-gray-500` |

### 影响范围

- ✅ 主页 `/`
- ✅ 文章详情页 `/article/[id]`
- ✅ 分类页 `/cate/[id]`
- ✅ 标签页 `/tag/[id]`
- ✅ 数据统计页 `/data`（固定背景图不变）
- ✅ 所有页面的 Footer
- ✅ 所有页面的 Header 导航

### 技术要点

- `Slide` 组件通过 `useConfigStore` 获取 `isDark` 状态，使用 `useMemo` 缓存背景图选择结果
- `StarrySky` 组件通过 `useConfigStore` 获取 `isDark` 状态，颜色作为 `useEffect` 依赖项触发 canvas 重绘
- 主题状态通过 `zustand` + `localStorage` 持久化
