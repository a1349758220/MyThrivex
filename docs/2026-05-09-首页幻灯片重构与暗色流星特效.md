### 1.2 MeteorShower 暗色流星效果

**核心文件**: [`ThriveX-Blog-3.3.6/src/components/Slide/components/MeteorShower.tsx`](ThriveX-Blog-3.3.6/src/components/Slide/components/MeteorShower.tsx:1)

全新 Canvas-less 流星组件，100% CSS 驱动：

- **流星颜色池**: 白、蓝、粉、橙、绿 5 色随机
- **运动轨迹**: 通过 CSS 自定义属性 `--x`、`--y`、`--angle`、`--distance`、`--length` 控制
- **双向飞行**: 50% 概率从左到右或从右到左
- **底部渐隐**: 通过 `requestAnimationFrame` 实时检测流星接近 Slide 底部时触发 `.home-meteor-shell--fade-out` 淡出
- **刷新机制**: 动画循环迭代时重新生成流星参数
- **每次仅渲染**: 1 颗流星，减少性能开销

配套 SCSS 动画：[`index.scss`](ThriveX-Blog-3.3.6/src/styles/index.scss:424) — `.home-meteor-layer`、`.home-meteor-shell`、`.home-meteor` 及其 `::before`/`::after` 光晕
