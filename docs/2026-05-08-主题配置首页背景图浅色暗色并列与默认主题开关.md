## 第三部分：简要说明

本变更对主题综合配置页面（[`SynthesisTheme/index.tsx`](../ThriveX-Admin-3.4.5/src/pages/setup/components/Theme/components/SynthesisTheme/index.tsx)）进行了如下调整：

1. **字段重命名**：原"首页背景图"标签改为"首页背景图（浅色）"，语义更清晰
2. **新增暗色背景图配置**：新增 `swiper_image_dark` 字段，用于存储暗色模式首页背景图 URL
3. **横向并列布局**：浅色/暗色两个背景图的输入框 + 图片预览通过 CSS Grid 实现 `lg:grid-cols-2` 响应式并列排布
4. **默认主题开关**：使用 `antd Segmented` 组件实现浅色/暗色一键切换，默认值为 `light`
5. **布局适配**：表单容器宽度从 `lg:w-[500px]` 扩展为 `lg:w-[900px]`，为并列布局提供足够空间
